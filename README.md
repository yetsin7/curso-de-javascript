## Para que sirve este curso

Este curso queda dedicado a JavaScript puro. La meta es que aprendas el
lenguaje con suficiente profundidad para trabajar en navegador, Node.js,
herramientas, DOM, asincronia, modulos y proyectos reales, sin depender de
frameworks para entender la base.

## Que aprenderas

- variables y scope;
- tipos de datos;
- operadores;
- estructuras de control;
- funciones;
- arrays y objetos;
- manejo de errores;
- asincronia;
- modulos;
- peticiones HTTP con `fetch`;
- DOM y eventos;
- Node.js y herramientas;
- testing, rendimiento y proyectos practicos.

## Limpieza de enfoque

Este repositorio ya no incluye bloques dedicados a TypeScript, React o Vue.
Esos temas viven mejor en sus propios cursos. Aqui la prioridad es dominar
JavaScript de verdad.
